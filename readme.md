my hyprland dotfiles on arch linux ^^

## info

- **os**: arch linux
- **compositor**: hyprland
- **terminal**: kitty
- **font**: Maple Mono NF + Cantarell
- **editor**: neovim
- **browser**: firefox + betterfox
- **app launcher**: rofi
- **notifications**: dunst
- **status bar**: waybar
- **lockscreen**: waylock

> drew a lot of inspiration (and copied a lot) from [eric murphy](https://github.com/ericmurphyxyz)
